## English
 Imagine a big city where traffic lights fail, bins overflow, and streetlights go out. Citizens call, complain, and wait, but the administration is overwhelmed. City Pulse acts as the "brain" of the city, constantly monitoring and reacting to these issues in real-time.Result: Problems are identified and flagged before they become public complaints.Why is it needed? To move from "reactive" city management to "proactive" management. It creates a seamless, efficient, and data-driven urban environment that improves the quality of life for every citizen.
### Tech Stack
- **Backend:** FastAPI, Python 3.10
- **AI/Agents:** Custom Agent Framework (.adk)
- **Deployment:** Google Cloud Run (Containerized via Docker)
- **Interface:** HTML5/CSS3 (Static Integration)
