## English
**City Pulse** is an AI-driven smart city platform designed to monitor and manage urban services efficiently. Utilizing intelligent AI agents and a robust FastAPI backend, it provides real-time insights and automated decision-making for urban infrastructure.

### Tech Stack
- **Backend:** FastAPI, Python 3.10
- **AI/Agents:** Custom Agent Framework (.adk)
- **Deployment:** Google Cloud Run (Containerized via Docker)
- **Interface:** HTML5/CSS3 (Static Integration)
